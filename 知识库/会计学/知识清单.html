第一章 总论

会计的定义与职能 必考

是什么：会计是以货币为主要计量单位，运用专门方法，对经济活动进行核算和监督的一种管理活动。--它不是简单的记账，而是一个完整的信息系统。

为什么需要会计：企业的利益相关者（投资者、债权人、政府、管理层）需要了解企业的财务状况和经营成果来做决策，而会计就是把纷繁复杂的经济活动翻译成标准化财务信息的语言。

两大基本职能：

  • 核算职能（反映）：对经济活动进行确认、计量、记录、报告 -- "把账算清楚"
  • 监督职能（控制）：对经济活动进行事前、事中、事后的监督控制 -- "看着别出错"
易错：核算和监督不是互斥的，核算是基础，监督贯穿全过程。考试常见陷阱是问"会计最基本职能是什么"--答案是核算（反映）职能。

会计基本假设 重点

动机：会计面对的是连续不断、币值波动的真实世界，必须设定一些前提条件才能开展工作。四大假设就是会计的"游戏规则"。

四大假设：

  • 会计主体假设：明确核算的空间范围--算谁的账，不能把老板个人的账和公司的账混在一起
  • 持续经营假设：假设企业将无限期经营下去--如果不假设持续经营，资产就不能按历史成本计价，而要按清算价值
  • 会计分期假设：把持续经营切成等距期间（年、季、月）--这是权责发生制的基础
  • 货币计量假设：以货币作为统一计量单位--隐含假设是币值稳定
易错：持续经营是会计分期的基础--先有持续经营，才需要分期。另外，货币计量假设隐含币值稳定，通胀严重时这一假设会受到挑战。

会计信息质量要求 重点

层次要求含义
首要质量可靠性如实反映，不偏不倚
首要质量相关性与使用者决策相关，具有预测价值和反馈价值
次要质量可理解性清晰易懂
次要质量可比性横向可比（不同企业间）+ 纵向可比（同一企业不同时期）
约束条件实质重于形式按经济实质而非法律形式处理（如融资租赁）
约束条件重要性重要事项详细反映，次要事项简化处理
约束条件谨慎性不高估资产和收益，不低估负债和费用
约束条件及时性不得提前或延后

第二章 会计要素与会计等式

六大会计要素 必考

为什么分六类：经济活动千变万化，但归纳起来无非三种类型--手里有什么（资产）、欠别人多少（负债）、自己剩多少（权益），以及赚了多少（收入）、花了多少（费用）、净得多少（利润）。

三大静态要素（反映财务状况，出现在资产负债表）：

  • 资产：过去的交易形成、企业拥有或控制、预期带来经济利益。关键词："过去的""控制的""未来利益"
  • 负债：过去的交易形成、预期导致经济利益流出、现时义务。关键词："现时义务"
  • 所有者权益：资产减负债后的剩余权益。关键词："剩余权益"--股东是最后分钱的人

三大动态要素（反映经营成果，出现在利润表）：

  • 收入：日常活动形成、导致权益增加、与投入资本无关。关键词："日常活动"
  • 费用：日常活动发生、导致权益减少、与分配利润无关。关键词："日常活动"
  • 利润：收入 - 费用 + 利得 - 损失。反映了企业一定期间的净增值
易错：收入和利得的区别在于"日常"还是"非日常"。卖产品是收入（日常），卖厂房是利得（非日常）。费用和损失同理。考试常考判断题--"企业所有经济利益的流入都是收入"是错的。

会计等式 必考

基本等式：资产 = 负债 + 所有者权益--这是静态等式，反映某一时点的财务状况。为什么相等？因为资产要么是借来的（负债），要么是股东投入或赚来的（权益）。

扩展等式：资产 = 负债 + 所有者权益 +（收入 - 费用）--这是动态等式。收入最终会增加权益，费用会减少权益，所以在期末结账前，等式要包含收入费用。

核心原理：任何经济业务的发生都不会破坏等式的平衡。这就是"双重影响"--每笔业务至少影响两个项目。

易错："资产增加必然伴随负债减少"是错的。资产增加可以同时使负债增加（借钱）、权益增加（股东投资）、或其他资产减少（用存款买设备）。

第三章 会计科目与账户

会计科目 重点

会计科目是对会计要素具体内容进行分类核算的项目。--如果把六大要素比作六个大类，会计科目就是每个大类下的明细分类。

五类科目记忆口诀：

  • 资产类：库存现金、银行存款、应收账款、存货、固定资产
  • 负债类：短期借款、应付账款、应交税费、长期借款
  • 所有者权益类：实收资本、资本公积、盈余公积、利润分配
  • 成本类：生产成本、制造费用、研发支出
  • 损益类：主营业务收入、管理费用、销售费用、财务费用

T 型账户与借贷方向 必考

动机：为什么要分借方和贷方？因为"资产=负债+权益"这个等式的左边和右边，增加的含义是相反的--资产增加在左边，负债和权益增加在右边。借贷记账法用"借"和"贷"来体现这种对称性。

账户类型借方（左）贷方（右）正常余额
资产类增加 +减少 -借方
负债类减少 -增加 +贷方
所有者权益类减少 -增加 +贷方
收入类减少 -增加 +期末无余额
费用类增加 +减少 -期末无余额
易错：资产+费用 是"借增贷减"，负债+权益+收入 是"贷增借减"。最常犯的错误是把费用类和损益类记反。记住口诀："资费借增，负权收贷增"。

第四章 复式记账

借贷记账法 必考

核心规则：有借必有贷，借贷必相等。--每一笔经济业务必须同时记入两个或两个以上账户，借方金额合计等于贷方金额合计。

会计分录三要素：记账方向（借/贷）、会计科目、金额。

试算平衡：

  • 发生额试算平衡：全部账户借方发生额合计 = 全部账户贷方发生额合计
  • 余额试算平衡：全部账户借方余额合计 = 全部账户贷方余额合计
易错：试算平衡只能检查借贷金额是否相等，不能发现所有错误。比如整笔漏记、科目用错、借贷同时多记少记，试算平衡都发现不了。

第五章 会计凭证与账簿

错账更正方法 重点

方法适用场景
划线更正法结账前发现账簿文字/数字错误（凭证没错，登账错了）
红字更正法凭证科目错误 或 金额多记
补充登记法凭证金额少记
易错：三种方法的区分关键是看"错在哪个环节"。凭证错了用红字或补充，账簿错了用划线。

第六章 财务报表

资产负债表 vs 利润表 必考

维度资产负债表利润表
性质静态报表（时点）动态报表（期间）
公式资产 = 负债 + 所有者权益收入 - 费用 = 利润
结构账户式（左资产右权益）多步式

利润表多步式结构：

营业收入 - 营业成本 = 毛利
毛利 - 税金及附加 - 期间费用 + 其他收益 = 营业利润
营业利润 + 营业外收支 = 利润总额
利润总额 - 所得税 = 净利润

财务比率 高频

类别指标公式
偿债能力流动比率流动资产 / 流动负债
偿债能力速动比率(流动资产 - 存货) / 流动负债
偿债能力资产负债率总负债 / 总资产
营运能力应收账款周转率销售收入 / 应收账款平均余额
营运能力存货周转率销售成本 / 存货平均余额
盈利能力ROA净利润 / 总资产
盈利能力ROE净利润 / 所有者权益
易错：速动比率剔除存货，因为存货变现能力最差。流动比率一般认为2比较安全，速动比率1比较安全--但不同行业差异很大。

第七章 主要经济业务核算

核心会计分录 必考

筹资：借：银行存款   贷：实收资本（接受投资）/ 短期借款（借款）

采购：借：原材料   应交税费-应交增值税（进项税额）   贷：银行存款

销售：借：银行存款   贷：主营业务收入   应交税费-应交增值税（销项税额）

同时结转成本：借：主营业务成本   贷：库存商品

利润分配：借：利润分配   贷：盈余公积 / 应付股利

易错：增值税是价外税，不进利润表。进项税额是"买的时候交的税"可以抵扣，销项税额是"卖的时候收的税"要上交。实际缴纳 = 销项 - 进项。
