第一章 GDP与国民收入核算

GDP核算 必考

支出法：GDP = C + I + G + (X-M) -- 消费+投资+政府购买+净出口。

生产法：GDP = 各行业增加值之和 = 总产出 - 中间投入。

收入法：GDP = 工资+利息+租金+利润+折旧+间接税-补贴。

名义 vs 实际：GDP平减指数 = 名义GDP/实际GDP。与CPI区别：平减指数涵盖所有国内产品（篮子可变），CPI固定一篮子消费品。

第二章 IS-LM模型

IS曲线 必考

产品市场均衡：Y = C(Y-T) + I(r) + G。利率上升 -> 投资下降 -> 乘数效应 -> Y下降。IS曲线向右下方倾斜。

移动因素：G增加或T减少 -> IS右移。乘数 k = 1/(1-MPC)。

LM曲线 必考

货币市场均衡：M/P = L(Y, r)。Y增加 -> 交易性货币需求增加 -> 利率上升。LM曲线向右上方倾斜。

流动性陷阱：利率极低时货币需求无限大，LM水平 -- 货币政策完全失效。

政策效果 重点

政策Yr挤出效应
扩张财政增加上升有（私人投资被挤出）
扩张货币增加下降无
易错：挤出效应大小取决于IS和LM斜率。LM越陡（货币需求对利率越不敏感）挤出越大；LM水平（流动性陷阱）时无挤出。

第三章 AD-AS模型

总需求曲线 重点

AD为何向下倾斜：(1)财富效应 (2)利率效应 (3)汇率效应。

总供给曲线 必考

类型形状假设政策含义
古典AS垂直价格工资完全灵活需求管理无效
凯恩斯AS水平价格工资刚性需求管理完全有效
常规AS右上倾斜短期价格粘性有效但伴有通胀

菲利普斯曲线 重点

原始PC：通胀与失业负相关。附加预期PC：pi = pie - beta(u-un) + v。短期存在权衡，长期PC垂直（自然率假说）。

第四章 经济增长

索洛模型 必考

核心方程：Delta k = sf(k) - (n+delta)k。s=储蓄率，n=人口增长率，delta=折旧率。

稳态：sf(k*) = (n+delta)k*。人均资本和人均产出不再增长。

黄金律：使消费最大化的k -- f'(k_gold) = n+delta。

易错：储蓄率提高 -> 新稳态人均收入更高，但仅过渡期有更高增长率。长期增长率 = 技术进步率（外生）。

内生增长 高频

AK模型：Y=AK，资本边际报酬不递减 -> 储蓄率影响长期增长率。机制：知识外溢、人力资本、R&D。

第五章 开放经济

蒙代尔-弗莱明模型 重点

汇率制度财政政策货币政策
浮动汇率无效（汇率升值挤出净出口）有效（汇率贬值增强净出口）
固定汇率有效（货币被动配合）无效（维持汇率约束）
蒙代尔不可能三角：固定汇率、资本自由流动、独立货币政策 -- 三者最多选两个。
