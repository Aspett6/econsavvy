第一章 货币与货币制度

货币层次 必考

层次内容
M0流通中现金
M1M0 + 企业活期存款
M2M1 + 准货币（定期+储蓄+其他存款）

货币乘数 重点

m = (1+c)/(r+c+e)，c=现金漏损率，r=法定准备金率，e=超额准备金率。

货币乘数不是固定值 -- 危机时银行增持超额准备金、公众增持现金，乘数大幅下降。

第二章 金融市场与利率

利率期限结构 重点

预期理论：长期利率 = 预期短期利率的几何平均。无法解释收益率曲线通常上倾。

流动性偏好理论：长期利率 = 预期短期利率 + 流动性溢价。解释了收益率曲线上倾。

有效市场假说：弱式（历史价格已反映）、半强式（公开信息已反映）、强式（内幕信息已反映）。

第三章 投资组合与CAPM

Markowitz模型 必考

核心：通过分散化降低非系统性风险。组合方差 = sum(wi*wj*Cov(i,j))。

有效边界：给定收益下风险最小的组合集。分离定理：引入无风险资产后，最优风险组合与风险偏好无关。

CAPM 必考

E(Ri) = Rf + beta_i[E(Rm)-Rf]，beta = Cov(Ri,Rm)/sigma_m^2。

CMLSML
适用范围仅有效组合所有资产
风险度量总风险sigma系统风险beta
CAPM只给系统性风险定价 -- 非系统性风险可通过分散化消除，不获得风险溢价。

APT与Fama-French 高频

APT多因子模型：E(Ri) = Rf + beta_i1*RP1 + beta_i2*RP2 + ...。Fama-French三因子：市场+规模(SMB)+价值(HML)。

第四章 期权定价

期权基础 重点

C = max(S-K, 0)，P = max(K-S, 0)。平价关系（欧式）：C + Ke^{-rT} = P + S0。

Black-Scholes 高频

通过构建无风险对冲组合推导。关键参数：S0、K、sigma、T、r。N(d1)近似为期权被执行的风险中性概率。

GammaDelta的变化率

第五章 风险管理与巴塞尔协议

VaR 重点

在置信水平alpha下持有期内的最大可能损失。方法：历史模拟、方差-协方差、蒙特卡洛。

VaR不满足次可加性 -- 组合VaR可能大于各部分VaR之和。CVaR满足次可加性。
